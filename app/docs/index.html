Registration


Username

When a user name is entered, it is checked whether it contains characters that cannot be written to the database. If this is the case, an error message is displayed above the input field, which tells the user which character he should remove.

Password

The password is first checked to see if it has eight digits. If this is the case, a request is sent to the API of haveibeenpwned.com whether the password is listed there or not. For this purpose, the first five characters of the SHA-1 hash are sent, the server returns the rest, which then will be compared to the hash of the password entered by the user.

The API offers a list of leaked passwords. Every password in this list has to be regarded as insecure, since we have to assume that a hacker also has access to these leaked passwords.
